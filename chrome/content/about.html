ChmSee 2.0

An HTML Help viewer for Unix/Linux.

Homepage: http://code.google.com/p/chmsee

Usage

If your favorite browser is Firefox, you will find ChmSee looks like a simple one. Instead of displaying web pages from internet, ChmSee let you to view HTML pages from compiled CHM file. ChmSee use the same xulrunner behind Firefox, so you can use the Firefox navigation commands and shortcuts in ChmSee.

  • File
    • Open file: Ctrl+O
    • Open tab: Ctrl+T
    • Close tab: Ctrl+W
  • Zoom
    • Zoom in: Ctrl+=
    • Zoom out: Ctrl+-
    • Zoom reset: Ctrl+0
  • Toggle panel: F9
  • Bookmark: Ctrl+D
  • Find: Ctrl+F
  • Print: Ctrl+P
  • About: F1
  • Quit: Ctrl+Q
